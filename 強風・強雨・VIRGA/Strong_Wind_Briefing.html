関東南西強風（SW STRONG WIND）── 3つのパターン

日本海低気圧の経路により、関東地方の南西強風は3パターンに分類される。NRT/HND運航に最も影響するパターンを理解することが重要。

1 低気圧が北海道の北を通過
  • 気圧傾度が緩やかで、地上風は比較的弱い
  • NRTでの影響は限定的。G/A・DIVは少数にとどまる
  • 850hPa風速は40-50kt程度
2 低気圧が日本海中部を通過 ── 最危険パターン
  • 関東平野全域でSW 40-50ktの地上風。NRT DIV多発パターン
  • 2013/3/13: NRTで1日50便以上のDIVを記録。G52kt観測
  • 850hPa: 70-80ktの下層ジェット → 地上換算で40-50kt
  • 低気圧中心と寒冷前線の位置関係がSW風を最大化
3 低気圧が北陸沿岸を通過
  • 風向がSSW寄りになり、NRT RWY16のXW成分が減少
  • パターン2に比べ運航影響は軽微だが、HND RWY22/23ではLLWSリスク残存
🌀

爆弾低気圧（BOMB CYCLONE）

  • WMO定義: 24時間で中心気圧が24hPa以上低下する温帯低気圧
  • 緯度補正: 24hPa × sin(φ)/sin(60°) → 北緯40°なら17.8hPa/24hが基準
  • 気象庁は「急速に発達する低気圧」と表現。JAL OCCでは社内慣例として「爆弾低気圧」を使用
  • 日本海で低気圧が発達 → 強い冬型気圧配置 → 全国的暴風
KEY INDICATOR: 850hPa面の風速50kt以上 → 地上は70%換算で35kt+G45ktを想定。850hPaで70kt超なら地上50kt級の暴風を覚悟。
◆

寒冷渦（COLD VORTEX）& ドライスロット

  • 寒冷渦: ジェット気流トラフが切離 → 独立した低気圧性渦。500hPa天気図で最も顕著
  • 南東象限で悪天発生 → 地上低気圧がこの位置で急発達
  • 500hPa上空に-24℃以下の強い寒気を伴うケースが多い
  • コンマ雲: 発達中の温帯低気圧の典型的雲パターン（,形）
  • ドライスロット: 乾燥寒気の貫入。出現すると低気圧は衰退期へ。ただし暗域付近で対流雲が急発達し局所的悪天をもたらす
注意: 寒冷渦は動きが遅いため、荒天が長時間持続する特徴あり。
⚙

ALWIN（空港低層風情報）── 活用のポイント

  • APT MODE: 空港5NM以内の代表風。GND~3000ftの風向風速・GUST
  • RWY MODE: 各滑走路の経路風。正対風(HW)・横風(XW)・GUST・Wind Shear情報
  • データ更新: 約3分ごと。ACARS版ではGUST時のみテキスト情報表示
  • SPD CHG表示 = 風速急変。MB = Microburst検出
  • GUST通報基準: 風速最大値と2分間平均値の差が10kt以上
  • HW GAIN/LOSS: 経路上の正対風変化量 → WS警戒の指標
  • GPWS Windshear Warning前に、ALWINでHW GUST・XW GUSTの出現を確認可能
  • Warning作動の10分前からGUST出現 → 事前察知に活用

空港個別の風特性と運航リスク
✈

CTS 新千歳 ── SE風パターン

  • 風向150°(SE)、2000ft 50kt以上でLLWS/MBAリスク
  • 太平洋側低気圧の接近時に発生。春〜秋に多い
  • 2015/10/1: G44kt(150°) → RWY01方向へのXW・TW増大
  • ALWIN RWY01L/Rの経路風とAPT MODEの代表風で差異が生じやすい → RWY MODEを優先確認
CTS SE風閾値: 2000ft 50kt以上 → LLWS/MBA警戒。ALWIN SPD CHG出現で即G/A検討。
✈

ITM 伊丹 ── 北風パターン

  • 台風・低気圧通過後の北風で強風発生
  • 六甲山系の風下側 → 山岳波・乱気流
  • RWY32L/R APP時: N風で2000ft付近にWS発生
  • 夏季の南風パターンとは異なるLLWSメカニズム
⚠

KMJ 熊本 ── 東風 + 山岳波

  • 東風50kt / 2000ftでMT.WAVE + LLWS発生
  • 阿蘇外輪山（標高800-1000m）を越えた東風が山岳波を形成
  • RWY07 APP時: 外輪山の風下でダウンドラフト → 急激な対気速度低下
  • 外輪山の上空通過時にMOD-SEV TURB報告多数
KMJ CRITICAL: 東風(080-100°) 2000ft 50kt以上 → 外輪山MT.WAVE + LLWS。RWY07 APP経路が直接影響を受ける。
⚠

KOJ 鹿児島 ── 東風 + 台地 + 谷

  • 空港東側の標高270mの台地と谷地形がLLWSを発生
  • 東風が台地を越える際にダウンドラフト → MBA/WSA
  • 谷を抜ける風がRWY34 APP経路で急激な風向変化を起こす
  • 地形性LLWS → ALWINでは捕捉困難な場合あり。PIREP活用
✈

OKA 那覇 ── 台風時の南東風 + RWY差異

  • 台風接近時: 風向130° → 次第に160°に変化。経路風と代表風に乖離
  • RWY18L ALWIN: 750ft 120°/31G42kt → GND 170°/22kt。上空と地上で50°の風向差
  • A-RWY vs B-RWY: B-RWYがA-RWYより2-4kt大きい風速を観測（台風5号事例）
  • 台風非暴風域でも3000ft G68ktを記録 → LLWS/XW SHEAR発生
✈

TKS 徳島 ── 南東風 + 低視程サークリング

  • 南東風パターンで「人」字型前線が形成される特殊気象
  • サークリングAPP中の急激な視程低下 → DVT多発
  • VOR/DME RWY29 Circling to RWY11 → SE風時にXW成分増大
  • 低視程+横風の複合リスク。ATIS/TAFの視程変化に要注意
