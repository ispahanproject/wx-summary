VIRGA（尾流雲）── HND冬季特有の現象

▶ 発生メカニズム

  • 降水粒子が地上に到達せず蒸発 → 蒸発冷却でドライマイクロバースト(MBA)発生
  • TAFでは予報されない。地上ではVMCのまま突然のWS/MBA
  • VIRGA → ドライマイクロバースト → MBA/WSA

▶ 発生条件（3要素）

  • ① 冬型気圧配置 + 上空寒気の残留
  • ② JPCZ → 北陸で降雪 → 関東は冬晴れ
  • ③ 丹沢山系で雲発生 → HND/房総半島方面へ流入

▶ 典型的パターン

  • 冬型 → 翌日晴れ → 昼過ぎにVIRGA発生
  • 地上: CAVOK/SKC。上空にちぎれ雲 → VIRGA
  • ALWINで低高度にG40ktが突然出現

▶ 確認方法

  • 気象解説情報のVIRGAコメントを確認
  • 時系列予想の-SHSN表記に注目
  • LFM(局地数値予報) 10000ft湿域チェック
  • レーダーで降水エコーが地上に達していない領域

HND VIRGA事例 ① 2011/1/26

  • 冬型気圧配置 → HND晴れ → 昼過ぎVIRGA発生
  • DVT 11便（JALグループ8便 + 欠航10便）
  • TAFでは全く予報されておらず、突発的に発生
  • ドライマイクロバーストにより着陸困難 → 大量G/A → DIV
IMPACT: TAF予報なし・地上VMCの中で11便DVT。VIRGA起源のドライMBAの危険性を示す典型事例。
⚠

HND VIRGA事例 ② 2015/2/13

  • 同様の冬型→晴れ→昼過ぎVIRGAパターン
  • 4便G/A。ALWIN低高度にG40ktを検出
  • 2011年事例の教訓から、ALWINデータの早期監視を実施
  • VIRGA発生を早期に検知し、2011年よりDIV数を抑制
LESSON: ALWINの低高度GUST出現をトリガーに、VIRGA警戒態勢を早期発動することで影響を軽減可能。

ALWIN活用フロー ── 強風判断

1 APT MODE確認
  • GND-3000ft風向風速の全体像を把握
  • SPD CHG / GUST表示の有無
  • MB Alert（Microburst検出）の確認
2 RWY MODE確認
  • 使用RWYの経路風(HW/XW)を確認
  • HW GAIN/LOSS量でWS強度を判定
  • XW GUST/SHEAR表示 → 横風急変警戒
3 意思決定
  • HW GUST + XW GUST同時 → WS Warning高確率
  • 10分間のトレンド変化を監視
  • PIREP・先行機情報と照合し最終判断
