降雪閾値(850hPa)
  • 日本海地方より降雪量は少ないが、じょう乱や冬型時に20cm超の大雪となることがある
  • 降雪の目安: 地上気温2℃以下 + 850hPa -9℃以下
  • 850hPa風が310-350° / 20kt以上で石狩湾CBが札幌方面から低地を通り流入 → しゅう雪
  • 比較的早朝の降雪は少なく、日中にかけて多い
⚡

大雪パターン — 3つの代表的気圧配置

① 強いNW冬型
  • 西高東低の冬型気圧配置
  • 北西風タイプ: 石狩湾からの流入で強い降雪
  • 西風タイプ: 山越えで弱い降雪
  • 地上気圧傾度が急 → 筋状雲 → 千歳流入
② 南岸低気圧
  • 津軽海峡南を通過 → 一般に雪
  • 発達時: E〜NE寒気移流で風雪強い
  • 千歳近傍通過＋発達 → 雨転リスク
  • 弱い低気圧の方が雪になりやすい
③ 石狩湾小低気圧 ⚠
  • 短時間に大雪の最危険パターン
  • NW季節風弱化 → 西岸でNW+NE気流収束 → 小LOW発生
  • 700hPaのNW風に乗って南下
  • 渦巻きレーダーエコーに要注意
⚠ 石狩湾小低気圧: 等圧線の間隔が広くても注意。雪雲が札幌経由でCTSに南下すると両RWY閉鎖に至る。コースがわずかにずれると降らない場合も（東側: 長沼～夕張方面 / 西側: 積丹～洞爺湖方面にそれる）。
◈

シヤーライン降雪

1月末〜2月
  • 500hPa: 満州寒冷低気圧とオホーツク海寒冷低気圧の間に東西シヤーライン
  • 地上のオホーツク海LOW中心からもシヤーラインが延びる
  • シヤーライン付近の収束が激しく、ゆっくり南下して千歳で大雪
◈

西海岸小低気圧

  • 冬型でNW季節風が弱まり始めた頃に発生
  • 北西季節風 + 内陸NE気流が西岸で収束 → 切離低気圧
  • 700hPaのNW風に乗って南〜SE〜E方向へ移動
  • 千歳に降雪をもたらす（石狩湾小低気圧と類似メカニズム）
●

雪あられ・氷あられ (GS)

  • 60%以上が地上気温-2℃〜+2℃で発生
  • 局地性が強く予想困難
  • TAFでTSSN表記時はGS発生の可能性を含む
  • 特別観測期間: 10/25〜4/30
  • 期間外はSPECIに表示されないため、整備担当者・PICの報告が重要

雷雨 (TS)

  • 西側の丘陵・羊蹄山で強制上昇 → CB発生
  • 700hPaの西風でCTSに流入しやすい
  • 発生目安: 850-500hPa温度差25℃以上 + 500hPaトラフ + 高湿数
  • 850hPa南風 → SW丘陵で発達したエコーが空港接近
  • TAFにCB付記の場合もTS可能性大 → 気象台に確認
  • ピーク: 7-8月、10月

その他特記事項

  • 年間を通じ南風の卓越比率が大きい
  • 観測計4個（地上7-8m高）で風を観測 — 地上と上層の風向差がMETARに反映されるのに時間差あり
  • Tailwind → Touchdown直前の風向変化に注意
  • 低層5,000ft以下で地形起因のTURBが特徴的
  • 空港南側の湿地帯の凸凹地形が乱気流の一因
  • 北方LOW型: 暖気移流が強まるとみぞれ・雨に変化
