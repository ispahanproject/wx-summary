霧の種類 — SIX TYPES OF FOG

霧は微小な水滴が空中に漂い視程1km未満となる現象。1km以上は「もや（BR）」。相対湿度はほぼ100%。複数要因の複合で発生することもある。

放射霧
RADIATION FOG
夜間の放射冷却で地表付近の空気が冷却。雲なし・弱風時に発生。秋〜冬、盆地・山間空港で多い。日射で消滅。
移流霧
ADVECTION FOG
冷たい海面上に暖湿気が流入し冷却。いわゆる海霧。梅雨期、北日本太平洋沿岸・瀬戸内海西部で多い。長時間持続。
蒸気霧
STEAM FOG
暖かい水面上を寒気が通過時に蒸発水蒸気が冷却。海水温が気温より6℃以上高い時に発生。冬の日本海・河川沿い。
滑昇霧
UPSLOPE FOG
地形斜面に沿って空気が上昇し断熱膨張で冷却。夏の山岳斜面で多い。標高の高い空港（TAK, HIJ等）で注意。
前線霧
FRONTAL FOG
前線面の暖かい雨が寒気中を落下し蒸発水蒸気が冷却。または雨で濡れた地面に冷気流入。広域・長時間継続。
混合霧
MIXING FOG
気温の異なる2つの空気塊が混合し暖気側が冷却。水温差のある河川合流点などで見られる。

黄砂・煙霧による視程障害

煙霧（HZ）: 乾いた微粒子による視程低下。METARでは視程5km以下で使用。
黄砂（SA）: 東アジア砂漠域から飛来。春に多い。地上視程より飛行視程が高いことが多く注意。

  • 寒冷前線通過で視程3,000mまで低下

ヤマセ 移流霧

瀬戸内海周辺の霧 — 3つの発生要因

① 放射霧

前日降水後に天気回復→夜間放射冷却で空気が飽和。4〜7月にOIT・KKJ・UBJ・MYJなど西部で発生。

② 移流霧（海霧）

太平洋高気圧縁辺の暖湿気が流入→冷たい海面で冷却。5〜7月、気温＞海水温となる時期に多発。風速5-6kt以下。

③ 前線霧

梅雨前線等の降水時、暖かい雨が冷気層中を落下し霧発生。広域・長時間。HIJ・OKJ・TAKなど高標高空港で顕著。

TAK（高松空港）— 悪天は「冬」にあり

梅雨期のイメージが強いが、統計的には1月がDIV最多（12件/5年）。2月10件、3月6件と冬季に集中。約3割がCAT N/G（予想困難な無情報DIV）。

  • 標高185m（607ft）の高台空港
  • 降水時の視程障害は予想困難
  • CIG 1000ft以下はFOGへの変身を「警戒」
  • 一気にマージナル→BLW WXの可能性

指標閾値適用備考

TAF確認のポイント — LESSONS LEARNED

TAF予報の限界

  • 霧の予報は数値予報でも困難
  • TAFにFG表現がなくても突然発生する事例が多数
  • MYJ 2013/4/25: TAF・気象官署見解とも霧情報なし→FG発生
  • CTS 2012/1/10: 前日TAFにFZFG予報なし→当日追加
  • TAK 2015/3/17: 早朝TAFにFG予報なし→AMD発行

要注意TAF表現

HND: 「TEMPO 1500 BR」→ 霧への発達リスク
TAK: CIG 1000ft以下 → FOGへの変身を警戒
OIT: 「BR」+弱い東〜南東風 → 海霧流入リスク
全般: 前日降水後の「NSW」→ 翌朝放射霧リスク

確認すべき情報

  • ひまわり霧情報（夜間含め30分毎）
  • FOSTER SYNOP視程情報
  • LFM下層悪天予想図
  • 海水温データ（神奈川県水産技術C等）

霧の解消パターン

霧種解消メカニズム所要時間注意事項
放射霧日射による気温上昇日の出後2-3時間秋は気温上昇弱く解消遅延。10-12月は昼まで残ることあり
移流霧（海霧）暖湿気供給の停止・風向変化午前10時頃まで弱い東〜南東風継続時は午後2時頃まで残存（OIT）
前線霧前線通過・風向変化終日継続の場合あり前線停滞時は長時間。気温20℃前後でも濃霧発生
蒸気霧風速変化・気温上昇日の出後2-3時間夕方以降発生分は風向・風速がキー（気温上昇望めず）
FZFG気温上昇・風向変化FOG BANKの移動次第FOG BANKの流入・移動は予測困難

運航判断のポイント — OCC DECISION MAKING

飛行計画作成時

  • 前日降水＋翌日晴天予報 → 放射霧リスク評価
  • 暖湿気流入パターン → 海霧・前線霧リスク評価
  • 瀬戸内海複数空港同時霧 → ダイバート先確保困難の認識
  • 高標高空港（TAK/HIJ/OKJ） → CIG低下即FOGの可能性
  • CAT-N/G PLNでも霧発生リスクあり → 燃料追加検討

運航中の監視

  • ひまわり霧情報・SYNOP視程をリアルタイム監視
  • 衛星可視画像でFOG BANK位置を確認（日中のみ）
  • METAR推移: PRFG→BCFG→FGの進行パターンに注意
  • 現地航務との情報交換（気象官署見解確認）
  • FOG BANKの移動方向・速度に注意（HND・CTS）
