CB タイプ

  • マルチセル型 ─ 日本で最も一般的。複数のセルが群として発達・衰退を繰り返す。個々のセル寿命は30-60分だが群全体として持続
  • スーパーセル型 ─ 単一の巨大セル。回転する上昇気流(メソサイクロン)を持つ。竜巻・大粒の雹・激しいダウンバーストを伴う

誘発雷 TRIGGERED LIGHTNING

最重要
  • 航空機の被雷の多くは「誘発雷」 ─ 電荷を帯びた雲域に航空機が接近・通過することがトリガーとなり放電が発生
  • 冬型の強風で雲頂付近の雲が流され、エコー中心から離れた場所でも電荷が蓄積 → 近くを通過する航空機が誘発
  • 現状の気象情報では誘発雷の予測・探知は困難。エコー域＋風下側CLDに対する十分な距離確保が最善策

冬季雷の発生条件

日本海側
  • シベリア寒気 → 暖かい対馬海流上で大気不安定
  • 500hPa気圧の谷の前面で発雷（谷通過後は止む）
  • 500hPa風向: WSW~SWで発生 → NWに変わると止む
  • 地上風向: SSW~SW 10kt前後（190°~230°方向）
  • 夕方~朝に多発、日中は比較的少ない
  • 継続時間: 1時間以内が半数、残りは平均4-5時間
  • あられ(GS)を伴う場合は特に発雷しやすい

冬季雷 被雷回避のポイント

  • 冬季の被雷はOAT 0℃~-5℃の高度帯（低層）で最も多い
  • DEP/ARR経路上のエコー域には十分注意。薄いエコーでも電荷蓄積あり
  • AXT事例: 最終進入前の低高度で山地付近の飛行を極力回避
  • 空港SOによる被雷層(0~-5℃)のBRIC入力 → VerticalとLateralに雷を回避
  • 被雷してAOGするくらいなら、DVTも選択肢のうちの一つ（着陸だけが正解ではない）

METAR TS通報の読み方

  • +TSRA の「+:Heavy」は降水強度がHeavyであることを示す（雷の強度ではない）
  • RMKS部のFBL / MOD / HVY TS は、LIDENの電流値をもとに通報者が推定情報として追記
  • HVY TS OHD = 空港内で雷電現象観測。実質的に地上作業中止・離着陸不可の状況が多い
  • ただし RMKS のみから「強度の雷雨 → 離着陸禁止」と直接判断すべきではない
📄

狭域悪天予想図 vs TAF

  • 狭域悪天予想図: LFMベース。TS予報域に注目
  • TAF: 基本的にMSMベース
  • LFMによる予報はTAFに反映されないことがあるため、狭域悪天予想図を別途確認することが重要
  • 2016/8/14事例: 雷発生確度「A」であったが、TAFには-TSRAの表現のみ
  • 2017/7/9事例: TAFではSHRAのみ予想 → 実際は雷雨に発展
時系列予報(雷発生確度) + 狭域悪天予想図(TS域) → イントラ気象サイトから参照可能

ダウンバースト / MBA

  • ダウンバースト: CBからの強烈な下降気流が地上付近で水平に拡散
  • 水平影響範囲 4km未満 = マイクロバースト
  • MBA (Microburst Alert)検出基準:
    最大風速差 15kt以上 + 影響面積 3km²以上
  • ITM事例: RWY32着陸側にMBA頻発 → 進入不可状態が継続
雷ナウキャスト活動度:
1 = 雷可能性あり / 2 = 雷あり / 3 = やや激しい雷 / 4 = 激しい雷
活動度3以上では積極的な回避行動を。
🛡

被雷回避策 AVOIDANCE STRATEGIES

ESSENTIAL
  • エコー + 風下側CLDの-10℃~0℃層では十分な距離を確保
  • Lightning Hit通報は後続機の被雷回避に極めて有効
  • APT INFO(QQA)の活用 ─ 被雷情報を共有
  • エコーを回避した後も薄い雲に電荷が蓄積している可能性に留意
  • 冬季: Vertical + Lateralに雷を回避（被雷層0~-5℃をBRIC入力）
  • KMQ: 小松管制隊に積極的にリクエスト（LIDAS情報豊富）
  • AXT: 最終進入前の低高度で山地付近飛行を回避
  • 性能が許せばRWY変更・DVTも選択肢（被雷AOG回避）

メガフラッシュ MEGAFLASH

  • WMO世界記録: 距離 709 km / 継続時間 16.73秒
  • 広域に渡る単一放電現象。通常の雷の概念を超えた距離に到達
  • CBから大きく離れた場所でも被雷リスクが存在する根拠
